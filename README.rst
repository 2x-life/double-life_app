#############
Double Life
#############


PC Installation
================

At the root of the projet run:

``$ pip3 install -e .``


To run the app use:

``$ python3 doublelife_gamebook/main.py``



Build Android APK
=================

At the root of the project run:
``$ buildozer -v android debug``

The APK file will be in the ``bin`` directory.
